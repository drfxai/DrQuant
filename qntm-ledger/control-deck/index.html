Issue & distribute

Mint to treasury

high-risk

Creates new QNTM into the treasury (debits genesis). Capped by QNTM_MAX_SUPPLY if set.

Grant to a user

high-risk

Moves issued QNTM from treasury to a user wallet.

Need a hard cap or a redemption path? Caps are set via QNTM_MAX_SUPPLY. There is deliberately no withdrawal / cash-out - QNTM flows in only.
